Проект на GitHub   Разработка сайта для онлайн магазина цветов

Описание: Вебсайт для заказа доставки цветов с базовой интеграцией заказов через телеграм-бота. Веб-приложение сделано с помощью фреймверка Django. В базах данных сохраняется информация о пользователях, о товарах и заказах. Вся информация о движении заказов автматически отправляется менеджеру в телеграм бот. Проект покрыт юнит-тестами

Проект на GitHub   Телеграм бот, интегрированный с локально развернутой LLM (большой языковой моделью)

Описание: Телеграм бот пересылает вопросы пользователя локально развернутой LLM и присылает в чат ответы модели. Контекст переписки сохраняется, бот помнит информацию из диалога с пользователем. Также бот умеет выполнять следующие команды:

  • /start — выводит приветствие и список доступных команд
  • /model — показывает название используемой LLM
  • /clear — очищает контекст переписки, бот забывает всю ранее полученную информацию
