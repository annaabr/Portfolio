Портфолио

Разработка сайта для онлайн магазина

Описание: Веб-приложение разработано с использованием Django.

Включает функции регистрации пользователей, просмотракаталога товаров с возможностью оставить отзыв на каждый товар, создания заказа.

Для администратора добавлена функция создания отчета по продажам за текущий день.
