Разработка сайта для онлайн магазина   Проект на GitHub

Описание: Веб-приложение разработано с использованием Django. Включает функции регистрации пользователей, просмотра каталога товаров с возможностью оставить отзыв на каждый товар, создания заказа. Для администратора добавлена функция создания отчета по продажам за текущий день.

Телеграм бот, интегрированный с локально развернутой LLM   Проект на GitHub

Описание: Телеграм бот пересылает вопросы пользователя локально развернутой LLM и присылает пользователю ответы LLM, а также выполняет команды: /start (выводится приветствие и список доступных команд) и /model (выводит название используемой LLM).

Игра-стрелялка   Проект на GitHub

Описание: Игра с постоянным передвижением, где на поле постоянно появляются монстры, которых нельзя касаться. Ведется подсчет и отображается количество убитых монстров и оставшихся жизней игрока.

Веб-приложение, интегрированное с внешним API   Проект на GitHub

Описание: Веб-приложение запрашивает случайные цитаты с публичного API и отображает их на странице. Использован ресурс.
